# ⚜️ Bangladesh Scouts - History & Heritage

![Bangladesh Scouts Banner](https://i.postimg.cc/59sL3zQK/scout-logo.webp)

> **স্বাধীনতার পর থেকে আজ পর্যন্ত—বাংলাদেশের বৃহত্তম যুব সংগঠনের গৌরবময় ইতিহাস ও ঐতিহ্যের পদযাত্রা।**

এই প্রজেক্টটি বাংলাদেশ স্কাউটসের ইতিহাস, নেতৃত্ব, কার্যক্রম, এবং সাংগঠনিক কাঠামো সুন্দরভাবে তুলে ধরার জন্য তৈরি করা হয়েছে। একটি আধুনিক, রেসপন্সিভ এবং ইন্টারঅ্যাক্টিভ ওয়েব এক্সপেরিয়েন্স দেওয়াই এই ওয়েবসাইটের মূল লক্ষ্য।

---

## ✨ Features (মূল বৈশিষ্ট্যসমূহ)

- 📜 **Interactive Timeline:** GSAP স্ক্রল ট্র্রিগারের মাধ্যমে স্কাউটিংয়ের ইতিহাস।
- 🗺️ **Dynamic Region Map:** বাংলাদেশের বিভিন্ন স্কাউট অঞ্চলের কাস্টম টুলটিপসহ ইন্টারঅ্যাক্টিভ ম্যাপ।
- 🔐 **Secure Admin Panel:** Firebase Authentication-এর মাধ্যমে সুরক্ষিত অ্যাডমিন ড্যাশবোর্ড।
- 🔄 **Real-time Data Update:** Firebase Cloud Firestore ব্যবহার করে ওয়েবসাইট নােমিয়েই ডাটা আপডেট করার সুবিধা।
- 🎨 **Modern UI/UX:** Tailwind CSS, গ্লাস মরফিজম (Glassmorphism), এবং কাস্টম কার্সরের সমন্বয়ে প্রিমিয়াম ডিজাইন।
- ⚡ **High Performance:** JSON ডাটা ফেচিং এবং অপ্টিমাইজড অ্যাসেটস।

## 🚀 Quick Setup (কীভাবে চালাবেন)

প্রজেক্টটি লোকালি রান করার জন্য নিচের ধাপগুলো অনুসরণ করুন:

1. **ফাইল ডাউনলোড করুন:** প্রজেক্টের জিপ ফাইলটি ডাউনলোড করে আনজিপ করুন।
   
2. **লাইভ সার্ভার চালু করুন:**
   VS Code এর "Live Server" এক্সটেনশন ব্যবহার করে `index.html` ফাইলটি ওপেন করুন।

3. **অ্যাডমিন প্যানেল অ্যাক্সেস:**
   ওয়েবসাইটের ডাটা পরিবর্তন করতে `/admin.html` এ গিয়ে আপনার ফায়ারবেস ক্রেডেনশিয়াল দিয়ে লগইন করুন।
