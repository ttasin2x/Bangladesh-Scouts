## 🎯 Project Overview (প্রজেক্ট সম্পর্কে)
এই প্রজেক্টটি বাংলাদেশ স্কাউটসের ইতিহাস, নেতৃত্ব, কার্যক্রম এবং সাংগঠনিক কাঠামো সম্পূর্ণ ডিজিটাল ও দৃষ্টিনন্দন উপায়ে তুলে ধরার জন্য তৈরি করা হয়েছে। একটি আধুনিক, রেসপন্সিভ এবং ইউজার-ইন্টারঅ্যাক্টিভ ওয়েব এক্সপেরিয়েন্স দেওয়াই এর মূল লক্ষ্য।

---

## ✨ Core Features (মূল বৈশিষ্ট্যসমূহ)

- 🚀 **Interactive GSAP Animations:** স্ক্রল করার সাথে সাথে টাইমলাইন ও অন্যান্য সেকশনে চোখ ধাঁধানো অ্যানিমেশন।
- 🗺️ **Dynamic Interactive Map:** বাংলাদেশের বিভিন্ন স্কাউট অঞ্চলের তথ্যসংবলিত কাস্টম টুলটিপ ম্যাপ।
- 🔐 **Secure Admin Dashboard:** Firebase Authentication-এর মাধ্যমে সম্পূর্ণ সুরক্ষিত অ্যাডমিন প্যানেল।
- 🔄 **Real-time CMS:** Firebase Cloud Firestore ব্যবহার করে ওয়েবসাইট না নামিয়েই লাইভ ডাটা আপডেট করার সুবিধা।
- 🎨 **Premium UI/UX:** গ্লাস মরফিজম (Glassmorphism), কাস্টম কার্সর এবং মডার্ন কালার প্যালেটের অপূর্ব সমন্বয়।
- ⚡ **Optimized Performance:** দ্রুত লোডিং স্পিড এবং লাইটওয়েট ডাটা ফেচিং।

## 📁 Project Architecture (ফাইলের গঠন)

```text
📦 Bangladesh-Scouts
 ┣ 📜 index.html         # মূল ওয়েবসাইটের ফ্রন্টএন্ড
 ┣ 📜 admin.html         # অ্যাডমিন লগইন এবং ড্যাশবোর্ড ইন্টারফেস
 ┣ 📜 style.css          # কাস্টম স্টাইলিং, গ্লাস মরফিজম এবং অ্যানিমেশন
 ┣ 📜 script.js          # লজিক, DOM ম্যানিপুলেশন এবং GSAP অ্যানিমেশন
 ┣ 📜 admin.js           # ফায়ারবেস অ্যাডমিন প্যানেল কন্ট্রোলার
 ┣ 📜 firebase-fetch.js  # ফায়ারবেস ডাটাবেস থেকে রিয়েল-টাইম ডাটা ফেচিং
 ┣ 📜 data.json          # স্ট্যাটিক ডাটা সোর্স (হিস্ট্রি, লিডারস, জাম্বোরি)
 ┗ 📂 photos/            # প্রজেক্টের সকল ছবি, লোগো ও গ্রাফিক্স
```
